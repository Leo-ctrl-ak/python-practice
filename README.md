# Python 基础语法练习

## 📌 项目简介
本项目用于练习 Python 基础语法，包含以下三个脚本：

1.  **guess_number.py**: 猜数字游戏（1-100），使用 `while` 循环和 `if` 判断。
2.  **fizzbuzz.py**: 打印 1-100，3 的倍数输出 Fizz，5 的倍数输出 Buzz。
3.  **reverse_string.py**: 定义函数实现字符串反转。

## 🚀 如何运行
进入 `day1-basics` 目录，在终端执行：
```bash
python guess_number.py
python fizzbuzz.py
python reverse_string.py
